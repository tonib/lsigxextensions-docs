Reordenar páginas de un tab control en un winform

Esta extensión permite reodenar de forma rápida las páginas de un tab control en un winform (Workpanel o Transaction).

Para poder usarla hay que abrir el objeto que contiene el tab control, y usar la opción del menú Lsi.Extensions > Forms > Reorder winform tab control pages. Aparecerá la siguiente ventana:

Tab control Aquí aparecen todos los tab controls del winform. Aparece la propiedad "Name" del control. Aquí se selecciona el control que se quiere modificar.
Tab pages Aquí aparecen las páginas del tab control seleccionado. El texto que aparece es la propiedad "Caption" de la página.
Move up / down Permiten subir o bajar la página seleccionada en "Tab pages".

Al confirmar la ventana se cerrará el objeto, se harán los cambios y se volverá a abrir el objeto con los cambios hechos en el tab control.

Limitaciones

  • Para poder usar la extensión el objeto no ha de tener modificaciones pendientes de guardar
  • No se pueden reordenar tabs que contengan páginas con captions repetidos
  • Si hay tab controls con la propiedad "Name" duplicada, la extensión puede fallar.
