Confirmación para Rebuild all

La operación Rebuild all en KBases grandes (ej. 10.000 objetos) puede llegar a tardar mucho tiempo (horas). Si se tiene la intención de hacer una operación Build all y por error se selecciona Rebuild all tienes un problema.

Esta extensión muestra un dialogo de confirmación cuando intenta hacer un Rebuild all o Rebuild main. Si el dialogo se cancela, la operación no se lanza:

Esta extensión se puede habilitar / deshabilitar en la configuración.
