Copiar objetos al portapapeles como tabla

La opción del menú Lsi.Extensions > Edit > Copy object info as table permite copiar información de los objetos actualmente seleccionados al portapapeles en formato tabular.

La selección se puede hacer en cualquier lugar de Genexus donde se puedan seleccionar objetos: En Work with objects, en la Folder view, etc. En caso de que la selección no se haga en una Tool Window de estas extensiones, se copia la siguiente información de cada objeto:

  • Nombre
  • Tipo de objeto
  • Descripción
  • Carpeta en la que está el objeto
  • Fecha de última modificación
  • Último usuario que ha hecho la modificación

Si la copia se hace en una lista de objetos en una Tool Window definida por estas extensiones, se copia la información mostrada en la lista. Esta información puede ser distinta para cada lista.
